macbook-brightness
==================

Mod para teclado luminoso de la Macbook Pro para la comunidad de Software Libre Programado en C/C++.

Compilar con el comando
=======================

$ make

Modo de uso
===========

./displaybrightness valor (opcional)

Donde 'valor' (float) va de 0 a 1.
